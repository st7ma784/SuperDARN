rl_forecast.reward module¶

rl_forecast.reward.compute_reward(x_last: Tensor, delta_pred: Tensor, y_true: Tensor, channel_weights: Tensor | None = None) → Tensor¶

Shaped reward for a single-step ionospheric delta prediction.

The reward is negative weighted MSE (obs channels masked to radar cells), plus physics-motivated penalties for spatial roughness and boundary drift.

Parameters:

  • x_last – (B, 6, H, W) most-recent normalised state frame

  • delta_pred – (B, 6, H, W) predicted delta (action in RL terms)

  • y_true – (B, 6, H, W) ground-truth next frame

  • channel_weights – optional (6,) override

Returns:

(B,) float tensor, detached from autograd graph

Return type:

reward

rl_forecast.reward.persistence_reward(x_last: Tensor, y_true: Tensor, channel_weights: Tensor | None = None) → Tensor¶

Reward of the trivial persistence forecast (predict no change).

rl_forecast.reward.reward_relative_to_persistence(x_last: Tensor, y_true: Tensor, channel_weights: Tensor | None = None) → Tensor¶

Reward = how much better the observed action is vs persistence (predict no change).

This gives meaningful signal in offline data where all observed actions were executed.
