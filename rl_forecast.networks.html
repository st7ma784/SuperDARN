rl_forecast.networks module¶

Neural network components for offline SAC ionospheric RL.

ConvEncoder — shared state encoder, (B, C, H, W) → latent + spatial features ActionEncoder — encodes a grid-space delta to a latent vector for the critic LatentActor — Gaussian policy in latent action space (reparameterised + tanh) GridDecoder — decodes (spatial features, latent action) → delta grid DoubleQCritic — twin Q-networks, (state_latent, action_latent) → (Q1, Q2)

class rl_forecast.networks.ActionEncoder(in_channels: int = 6, action_latent_dim: int = 128, pool_size: int = 8)¶

Bases: Module

Encodes a full-grid delta (B, 6, H, W) into a latent vector (B, action_latent_dim). Used to represent data-set actions in the critic’s input space. Lightweight: AdaptivePool + MLP, no transposed convolutions needed.

Initialize internal Module state, shared by both nn.Module and ScriptModule.

class rl_forecast.networks.GridDecoder(feat_channels: int = 256, action_latent_dim: int = 128, out_channels: int = 6, base_channels: int = 64)¶

Bases: Module

Decodes (spatial_features, latent_action) → (B, 6, H, W) delta grid.

spatial_features comes from ConvEncoder. latent_action is the actor sample. The action is spatially broadcast and concatenated with the features before upsampling, so it modulates the decode at every spatial location.

Initialize internal Module state, shared by both nn.Module and ScriptModule.
